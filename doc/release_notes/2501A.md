# Release Notes - Version 2501A

## Major Changes

### Ruby Upgrade
- Upgraded to Ruby 3.2.2
- Updated gem dependencies for Ruby 3 compatibility

### Features
- **Special Offer Codes**: Added ability to set special offer codes via URL parameter **spofrcode**
  - Clears special offer cookie after first use or two days, whichever comes first
  - Improved cookie handling and naming conventions
- **Referral System**: Added ability to set marketing source via URL parameter **referral_code**
  - Marketing source field now hidden when referral code is present
  - Aligned cookie naming with referral_code convention
  - Referral codes persist for 2 days

### Documentation
- Added project documentation structure
- Created release notes directory for tracking features and bug fixes
- Started comprehensive documentation for stagemgr
