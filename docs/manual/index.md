# Stagemgr User Manual

Stagemgr is a ticketing platform for live theater venues. It handles ticket sales, reserved seating, memberships, flex passes, donations, reporting, and all back-office operations for box office and theater management.

This manual covers everything you need to set up, configure, and operate Stagemgr on a daily basis.

## Choose Your Starting Point

### I'm setting up Stagemgr for a new theater

Start with [Setup & Configuration](setup/theaters.md) to create your theater, venues, seat maps, and payment types. You'll also want to review [Core Concepts](getting-started/concepts.md) to understand how Stagemgr's data model works.

### I'm an administrator managing productions

Start with [Creating a Production](productions/creating-a-production.md) to learn how to set up shows, configure performances, and manage pricing. See [Offers & Pricing](offers/special-offers.md) for promotions, flex passes, and memberships.

### I'm box office staff selling tickets

Start with [Ticket Order: General Admission](ticketing/creating-ticket-order-ga.md) or [Ticket Order: Reserved Seating](ticketing/creating-ticket-order-rs.md) depending on your venue. See [Order Lifecycle](ticketing/order-lifecycle.md) for an overview of how orders move through the system.

### I need to run reports or import data

See [Reports](reports/reports-overview.md) for all available reports, or [Imports](imports/imports-overview.md) for bulk data import procedures.

## Quick Reference

| Task | Where to Go |
|------|-------------|
| Create a new ticket order | **Orders** menu or **New Ticket Order** button |
| Find a production across all theaters | **Productions** menu |
| Look up a customer | **Customers** menu |
| View upcoming house counts | **My Account** (dashboard) |
| Run a sales report | **Reports** menu |
| Compare show performance | **Analysis** menu |
| Set up a special offer | **Options > Special Offers** |
| Group shows into a festival | **Options > Festivals** |
| Clone an existing special offer | Edit the offer, click **Clone as New Offer** |
| Import mailing list data | **Options > Import data** |
| Manage staff users | **Options > Administer Users** |
| Configure payment types | **Options > Manage Payment Types** |
| Create a seat map | **Options > Venues**, then select a venue |
